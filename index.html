Peralatan Presisi & Database CNC

Panduan Setup Supabase Sync

Supabase = database PostgreSQL gratis, REST API native, tidak ada CORS issue. Setup sekali, tidak perlu deploy ulang.

1Buat Akun & Project Supabase

Daftar gratis di supabase.com → New Project → isi nama & password database → tunggu project siap (~1 menit)

2Buat Table "offset_db"

Di dashboard Supabase → SQL Editor → klik New query → paste SQL berikut → klik Run:

-- TABLE 1: Data offset CREATE TABLE IF NOT EXISTS offset_db ( id text PRIMARY KEY, line integer NOT NULL, type text, op text, tool text, x numeric DEFAULT 0, z numeric DEFAULT 0, updated_at timestamptz DEFAULT now() ); ALTER TABLE offset_db ENABLE ROW LEVEL SECURITY; CREATE POLICY allow_select ON offset_db FOR SELECT USING (true); CREATE POLICY allow_insert ON offset_db FOR INSERT WITH CHECK (true); CREATE POLICY allow_delete ON offset_db FOR DELETE USING (true); CREATE POLICY allow_update ON offset_db FOR UPDATE USING (true) WITH CHECK (true); -- TABLE 2: Config aplikasi (password, kode koneksi, dll) CREATE TABLE IF NOT EXISTS app_config ( key text PRIMARY KEY, value text ); ALTER TABLE app_config ENABLE ROW LEVEL SECURITY; CREATE POLICY cfg_select ON app_config FOR SELECT USING (true); CREATE POLICY cfg_insert ON app_config FOR INSERT WITH CHECK (true); CREATE POLICY cfg_delete ON app_config FOR DELETE USING (true); CREATE POLICY cfg_update ON app_config FOR UPDATE USING (true) WITH CHECK (true);
3Ambil Project URL & Anon Key

Di Supabase dashboard → Project Settings → API → salin:
• Project URL (format: https://xxxx.supabase.co)
• anon public key (string panjang)

4Paste di Admin Panel

Di Admin Panel → isi Supabase URL dan Anon Key → klik Koneksikan. Selesai!
